1 Задача

У ковбоя Джо есть набор карт и желаемый набор карт, при котором он выигрывает
Джо может выбрать какой-то отрезок в своем списке карт и отсортировать его не по убыванию
Если он в итоге получит желаемый набор карт, то он победил

Входные данные: Первая строчка - количество карт
                Вторая строчка - набор карт Джо
                Третья строчка - Желаемый набор карт

Выходные данные: Вывести YES, если получился желаемый набор и NO, если нет

Пример: Набор Джо - 1 2 3 6 5 4 7
        Желаемый набор - 1 2 3 4 5 6 7
        Ответ - YES

2 Задача

Шериф дал бумажку ковбою Джо в которой есть набор букв, Джо необходимо посчитать,
сколько из этих букв можно собрать слово sheriff (каждая буква участвует только один раз)

Входные данные: Строка со строчными буквами без разделения

Выходные данные: Вывести сколько слов может получиться

3 Задача

Ковбой Джо захотел ограбить банк, но на определенную сумму. В банке лежат купюры разных
номиналов, но купюры каждого номинала только две. Помогите Джо узнать, может ли он
ограбить банк

Входные данные: 1-я строчка - Два числа: сумма, которую хочет украсть Джо и количество номиналов купюр
                2-я строчка - Номиналы купюр

Выходные данные: Вывести количество купюр, которые украдёт Джо, а на следующей строке
                 номиналы этих купюр, если вариантов несколько - вывести любой
                 Если ограбить не получится, вывести -1

4 Задача

Помогите Джо купить револьвер, у него есть определенная сумма денег, а в магазине есть n-е количество
револьверов по определенной цене

Входные данные: 1-я строчка - Два числа: количество револьверов и бюджет Джо
                2-я строчка - Стоимость револьверов через пробел

Выходные данные: Вывести цену револьвера с максимальной ценой, которую может себе позволить Джо
                 Если Джо не может купить ничего, то ответ 0
